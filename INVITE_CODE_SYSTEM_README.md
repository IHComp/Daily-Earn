# نظام أكواد الدعوة - Invite Code System

## نظرة عامة
نظام أكواد الدعوة يمنع الوصول إلى الموقع إلا بإدخال كود دعوة صحيح. النظام يعمل بالكامل على GitHub Pages (لا يحتاج PHP).

## المميزات
- ✅ واجهة Blur جميلة تغطي الموقع بالكامل
- ✅ أكثر من 150 كود دعوة جاهز
- ✅ كل كود يمكن استخدامه مرة واحدة فقط
- ✅ حفظ الجلسة - لا يحتاج المستخدم لإدخال الكود مرة أخرى
- ✅ يعمل بالكامل على GitHub Pages (HTML/CSS/JavaScript فقط)

## كيفية إضافة أكواد دعوة جديدة

افتح ملف `js/invite-code-system.js` وابحث عن المصفوفة `INVITE_CODES`:

```javascript
const INVITE_CODES = [
    'INVITE001', 'INVITE002', 'INVITE003',
    // ... أضف أكواد جديدة هنا
    'YOUR_NEW_CODE_1', 'YOUR_NEW_CODE_2'
];
```

أضف أكواد جديدة إلى هذه المصفوفة. يمكنك استخدام أي تنسيق تريده للأكواد.

## كيفية عرض الأكواد المستخدمة

افتح Console المتصفح (F12) واكتب:

```javascript
InviteCodeSystem.getUsedCodes()
```

سيظهر لك قائمة بجميع الأكواد المستخدمة مع أسماء المستخدمين وتواريخ الاستخدام.

## كيفية تسجيل الخروج (للتطوير)

افتح Console المتصفح (F12) واكتب:

```javascript
InviteCodeSystem.logout()
```

سيتم حذف الجلسة الحالية وإعادة عرض واجهة الدخول.

## كيفية الحصول على اسم المستخدم الحالي

افتح Console المتصفح (F12) واكتب:

```javascript
InviteCodeSystem.getUserName()
```

## الملفات المضافة
1. `js/invite-code-system.js` - ملف JavaScript الرئيسي للنظام
2. `css/invite-overlay.css` - ملف CSS لواجهة Blur
3. تعديلات على `index.html` - إضافة الواجهة وربط الملفات

## ملاحظات مهمة
- الأكواد المستخدمة تُحفظ في `localStorage` للمتصفح
- كل متصفح لديه قائمة منفصلة من الأكواد المستخدمة
- إذا حذف المستخدم بيانات المتصفح (localStorage)، يمكنه استخدام الكود مرة أخرى
- النظام يعمل على GitHub Pages بدون أي مشاكل

## الأكواد الافتراضية
النظام يأتي مع 150 كود افتراضي:
- INVITE001 إلى INVITE150

يمكنك تغييرها أو إضافة أكواد جديدة حسب احتياجك.
